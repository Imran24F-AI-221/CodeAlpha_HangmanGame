# CodeAlpha Python Internship

## Task 1 - Hangman Game

### Description
This is a simple text-based Hangman Game developed in Python.

### Features
- Random word selection
- 5 predefined words
- Maximum 6 wrong attempts
- Input validation
- Duplicate letter checking
- Win/Lose messages

### Technologies Used
- Python 3

### Concepts Used
- random module
- while loop
- if-else
- lists
- strings
- user input

### How to Run

python hangman.py
